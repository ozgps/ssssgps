install software:

1. Click the download link above to download the software.
2. After the software download is successful, open "Settings" -> "General" -> "Device Management", open the certificate file in the list, and click "Trust".
3. Insert the device and open the SSSS software.

Positioning mode:

Select the first icon in the upper right corner, move the map, or search for the location you want to locate from the search bar above, and click "Lock Coordinates".

Cruise mode:

1. Select the second icon in the upper right corner, set the route you want to simulate by clicking "Add Node", and then click "Start Planning".
2. Parameter setting: Average speed - the average speed of the cruise. Dynamic Speed ​​- The magnitude of random acceleration and deceleration in motion. Traffic lights - Waiting for traffic lights.
3. Click "Start Mobile".

Precautions:

1. If the software cannot be opened during use, please try to uninstall the software and download it from the “Alternate Address” next to the download button.
2. If you want to keep positioning, or keep cruising, you need to keep the device and mobile phone connected. Once the device is unplugged, the GPS of the mobile phone will return to the real position within a few seconds to several minutes.
3. This device supports iOS9.0 or above mobile phone, but there is insufficient support for 11.0.x system. If your mobile phone is 11.0.x, please upgrade to the latest system.
4. If you have other questions during use, you can consult your sales representative.
